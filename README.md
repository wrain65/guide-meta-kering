# guide-meta-kering

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖技术文档、参考指南、学习笔记等。这些页面以静态 HTML 形式存储，可直接在浏览器中打开查看，无需额外依赖或服务端支持。

该仓库不针对任何特定网站或域名，仅作为个人或团队内部的知识积累与归档用途。

## 目录说明

- `pages/` — 存放所有 HTML 页面文件，按主题或日期分类。
- `assets/` — 存放页面中引用的图片、样式表（CSS）、JavaScript 等资源文件。
- `index.html` — 仓库的页面索引，方便快速浏览所有归档页面。

## 页面归档说明

所有页面均为独立 HTML 文件，确保在没有网络连接的情况下仍可本地打开使用。归档内容不定期更新，可能包括：

- 技术参考与速查表
- 项目笔记与总结
- 学习路线图与教程
- 其他有价值的静态文档

页面命名遵循 `[主题]-[版本或日期].html` 的格式，便于识别与检索。

## 维护说明

本仓库由个人维护，欢迎通过 Issues 提交问题或建议。如果你希望贡献内容，请先阅读以下规则：

- 页面应为独立 HTML 文件，尽量不使用外部资源链接（如有，请一并放入 `assets/` 目录）。
- 避免在页面中包含任何形式的推广、营销或诱导点击内容。
- 提交前请确保页面在主流浏览器中正常渲染。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接双击 `pages/` 目录下的 HTML 文件，即可在浏览器中查看。
3. 也可通过 `index.html` 浏览所有归档页面的列表。
